# 🛒 MERN Stack E-Commerce Project

This is a full-stack e-commerce web application built using the **MERN stack** (MongoDB, Express.js, React.js, and Node.js). It allows users to browse products, add them to the cart, and place orders, while admins can manage products, orders, and users.

---

## 🚀 Features

### 👥 Users
- Register and login
- View products and details
- Add to cart and checkout
- View order history

### 🛠️ Admin
- Add, update, or delete products
- View and manage all orders
- Manage users

---

## 🧰 Tech Stack

- **Frontend**: React.js, React Router, Axios, Tailwind CSS (or Bootstrap)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT & bcrypt
- **Storage**: Multer for image upload (if used)
- **Payment**: (Optional) Stripe / Razorpay integration
